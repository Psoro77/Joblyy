# Joblyy 🤖💼

**Joblyy** is an AI-powered job application assistant. Give it a job link or a job description, and it handles the rest — from searching for matching opportunities to applying automatically on your behalf.

---

## ✨ Features

- **Link or description input** — Paste a job posting URL or type a description of the role you are looking for.
- **AI-powered job search** — The AI finds job listings that match your criteria across multiple platforms.
- **Automatic applications** — Once a match is found, Joblyy applies to the position for you automatically.

---

## 🚀 How It Works

1. Open the Joblyy interface.
2. Enter one or more job links **or** describe the type of job you are looking for (title, skills, location, etc.).
3. The AI analyzes your input, searches for relevant openings, and submits applications on your behalf.

---

## 🛠️ Tech Stack

- **Frontend** — Web interface for entering job links and descriptions.
- **AI / LLM** — Large Language Model that interprets job descriptions, searches for listings, and fills out application forms.
- **Automation** — Browser automation layer that navigates job portals and submits applications.
